# 参数使用指南

这个工具用物理模型模拟胶片的完整成像过程，不是套滤镜。
下面按影响力从大到小说明哪些参数真正影响你的照片。

---

## 第一层：决定整体色彩性格

**主面板 → 胶片与相纸**

这是最大的决定。不同胶片的色彩性格来自三个化学层的光谱敏感曲线和染料耦合关系，不是靠调色调出来的。

| 你想要的感觉 | 选这个胶片 |
|------------|----------|
| 暖调人像，高光柔和，阴影偏绿 | kodak_portra_400 |
| 极高饱和，风光，正片感 | fujifilm_velvia_100 |
| 电影感，日光场景 | kodak_vision3_50d |
| 电影感，室内钨丝灯 | kodak_vision3_500t |
| 日系清冷，阴影偏绿蓝 | fujifilm_pro_400h |
| 复古暖调，Kodachrome 经典 | kodak_kodachrome_64 |
| 高饱和风光，细腻颗粒 | kodak_ektar_100 |

相纸推荐起点：`kodak_ultra_endura` 或 `fujifilm_crystal_archive_typeii`，电影感用 `kodak_2383`。

## 第二层：曝光和色调（每次都要调）

**主面板 → 曝光控制 / 印刷色调 / 色调**

### 曝光补偿 EV

胶片有自己的最佳曝光区间，不是数码的 0EV 就是最好。

| 胶片 | 建议曝光 | 原因 |
|------|---------|------|
| Portra 系列 | +0.5 到 +1EV | 高光更柔，肤色更暖 |
| Velvia | -0.3EV | 饱和度更高 |
| Vision3 系列 | 0EV | 正常曝光即可 |

### 印刷色调

模拟暗房里调整放大机滤镜，是控制整体色调最直接的手段。

| 滑杆 | 向左 | 向右 |
|------|------|------|
| 暖 ←→ 冷 | 偏暖偏黄 | 偏冷偏青 |
| 品红 ←→ 绿 | 偏品红 | 偏绿，日系感 |

典型范围 ±30，超过 ±40 会比较夸张。

### 色调（胶片 gamma / 相纸 gamma）

**主面板 → 色调**

- `胶片 gamma 系数`：控制负片密度曲线的对比度。>1 对比更强，<1 更柔。默认 1.0 是物理真实值。
- `相纸 gamma 系数`：控制相纸的对比度响应。两个 gamma 叠加，是整体反差的核心控制。

> 先定 gamma，再调印刷色调，顺序不要反。

## 第三层：Halation（胶片感最强的特征）

**效果 Tab → Halation（红光渗透）**

Halation 是胶片最具辨识度的特征——红光穿透乳剂层反射回来，在高光边缘产生红色/橙色光晕。数码完全没有这个效果。

### 常用参数

- **散射强度**：光在乳剂内部扩散，让高光边缘轻微发散，细节稍微柔化。这是胶片"不那么锐利"的来源。
- **Halation 强度**：背反射光晕的总量。1.0 是物理默认值（最真实），调高会让胶片感更强。

| 想要的效果 | 散射强度 | Halation 强度 |
|----------|---------|-------------|
| 最真实（物理默认） | 1.0 | 1.0 |
| 更强的胶片感 | 1.2–1.5 | 1.2–1.5 |
| 干净一点 | 0.5–0.8 | 0.5–0.8 |
| 完全关闭 | 0 | 0（或关闭开关） |

> 其他 Halation 参数（散射核心、尾部、反弹次数等）是物理底层参数，通常不需要动。作者已按胶片数据设好了默认值。

## 第四层：颗粒（质感）

**效果 Tab → 颗粒**

这里的颗粒不是普通噪点叠加，是基于卤化银颗粒物理模型的光谱级模拟，颗粒大小、分布、染料云扩散都来自真实胶片测量数据。

### 颗粒大小（最重要）

| 颗粒大小 | 对应感觉 |
|---------|---------|
| 0.1 | ISO 100，极细腻 |
| 0.4 | ISO 400，标准 |
| 0.8 | ISO 800，明显颗粒 |
| 1.5 | ISO 3200，粗粒 |

### 模糊 sigma（像素）

控制颗粒的"软硬"。0.6–0.7 是标准值，高分辨率图像可以调到 0.8–0.9，让颗粒看起来更自然。

> 颗粒在预览模式下不显示，需要全分辨率扫描才能看到实际效果。

> 其他颗粒参数（RGB 层颗粒比例、子层缩放、均匀度等）是微调，通常不需要动。

---

## 第五层：色彩交叉（高级）

**效果 Tab → 色彩交叉（高级）**

这是"为什么 Portra 和 Pro 400H 颜色性格不同"的根本原因。切换胶片型号时会自动加载该胶片的物理默认值，通常不需要手动调。

如果想实验：

| 参数 | 调低效果 | 调高效果 |
|------|---------|---------|
| 全局倍率 | 色彩更"干净"，层间串扰减少 | 色彩关系更夸张 |
| 跨层抑制 | 饱和度降低 | 饱和度增强 |

---

## 第六层：效果细节

**效果 Tab**

### 眩光

模拟相纸表面的光泽反射。默认开启，强度 0.03 是很微弱的效果，通常不需要改。

### 预闪

暗房技术：在正式曝光前给相纸一次极低剂量的均匀曝光，压缩高光、提亮阴影，降低整体对比度。想要更低对比度时用，曝光量 0.01–0.05 EV。Y/M 滤镜偏移可以给预闪单独调色温。

### 前期柔光

模拟拍摄时镜头前加的柔光镜（Pro-Mist 等），作用在胶片曝光之前。影响整体柔化感。

### 后期柔光

模拟暗房放大机镜头前加的柔光镜，作用在印放阶段。和前期柔光叠加使用可以做出不同质感。

## 高级参数（通常不需要动）

**高级 Tab**

### RAW 参数

只在导入 RAW 文件时生效。白平衡默认 `as_shot`（使用相机记录的白平衡），对专业摄影师已经是正确答案。镜头校正需要相机和镜头在 lensfun 数据库里有记录。

### 输入图像

- **输入色彩空间**：默认 `ProPhoto RGB`，对应 Darktable 线性导出。如果导入普通 JPEG，改为 `sRGB` 并开启 CCTF 解码。
- **应用 CCTF 解码**：ProPhoto RGB 线性不需要开，sRGB JPEG 需要开。

### 光谱上采样

控制 RGB 到光谱的转换方法。`hanatos2025` 支持完整可见光域，是默认值，通常不需要改。

## 推荐工作流

### 快速出片（5 分钟）

1. 导入图像
2. 选胶片型号
3. 调曝光补偿 EV
4. 调印刷色调（暖冷 / 品红绿）
5. 调 gamma 系数（对比度）
6. 预览，满意就保存

### 精细调整

| 问题 | 解决方法 |
|------|---------|
| 高光太硬 | 调高 Halation 强度 |
| 颗粒太假 | 调整颗粒模糊 sigma |
| 色彩关系不对 | 调色彩交叉全局倍率 |
| 想要更暗房感 | 开预闪，曝光量 0.02–0.05 |
| 整体太平 | 提高胶片/相纸 gamma 系数 |
| 整体太硬 | 降低 gamma，或开预闪 |

### 不需要动的参数

散射核心/尾部、Halation 反弹次数/衰减比、颗粒各层缩放、光谱上采样参数——这些是物理底层，作者已按胶片数据设好了。
