## 一、项目定位

这是 spektrafilm（原 agx-emulsion）的换皮版本。

**不改变的：** 物理算法、pipeline 逻辑、胶片数据
**改变的：** 界面语言、视觉风格、交互体验、工作流顺序

## 三、Pipeline 顺序（不可打乱）

```
RAW/RGB 输入
  ↓ [相机阶段 - FilmingStage.expose()]
  光谱上采样、曝光补偿、前期柔光、镜头模糊、Halation
  ↓ [胶片显影 - FilmingStage.develop()]
  密度曲线、DIR 耦合剂、颗粒（在负片密度空间）
  ↓ [印放阶段 - PrintingStage]
  放大机曝光（Y/M 滤镜）、相纸显影、预闪、后期柔光
  ↓ [扫描阶段 - ScanningStage]
  白点/黑点校正、输出锐化、眩光
  ↓ 输出
```

## 四、Preview Mode 的影响

`params_builder.py` 里，`preview_mode=True` 时强制关闭：
- `grain.active = False`（颗粒）
- `film_render.halation`（Halation，通过 bypass）
- `enlarger.lens_blur = 0`
- `dir_couplers.diffusion_size_um = 0`

**这是作者的设计，不是 bug。** 颗粒在低分辨率预览图上物理尺寸失真。

当前处理方式：效果 Tab 里有颗粒裁剪预览窗口（小地图 + 300×300 全分辨率裁剪），调颗粒参数松手后自动刷新。

### Bug 修复（已完成）

| Bug | 根因 | 修复 |
|---|---|---|
| 对比模式按 Y 切原图全黑 | uint8 占位图锁了 contrast_limits=(0,255)，真图 float32 被渲染成黑 | 占位图改 float32 + 强制 contrast_limits=(0,1) |
| 颗粒大小=0 崩溃 | `grain.py` 除以零 | `params_mapper._apply_grain` 检测 ≤0 时关闭颗粒 |
| 胶片/相纸 gamma=0 崩溃 | `density_curves.py` 除以零 | `params_mapper._apply_special` + `widget_specs` 下限 0.01 |
| 白点≤黑点 崩溃 | `color_reference.py` m=0 除以零 | `params_mapper._apply_scanner` 强制 white > black |

## 七、params_mapper 映射规则

`params_mapper.py` 是 GUI state → runtime params 的唯一桥梁。

**单位换算：**
- `halation_strength` / `scatter_tail_weight`：GUI 存 0-100 百分比，mapper 除以 100

**崩溃保护（三处）：**
- 颗粒大小 ≤0 → 关闭颗粒
- gamma ≤0 → 强制 0.01
- 白点 ≤ 黑点 → white = black + 0.01

**硬编码的性能参数：**
```python
params.settings.use_enlarger_lut = True
params.settings.use_scanner_lut = True
params.settings.lut_resolution = 17
params.settings.use_fast_stats = True
```
